# Backend - Sistema de Gestión de Estadías

Este es el servidor backend para el sistema de estadías. Construido con Node.js y Express.

## Configuración Inicial

1.  Abre una terminal en esta carpeta.
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor en modo desarrollo:
    ```bash
    npm run dev
    ```

El servidor correrá en `http://localhost:3000`.

## Estructura

-   `server.js`: Punto de entrada.
-   `src/routes`: Definición de rutas (API endpoints).
-   `uploads`: Carpeta donde se guardarán los archivos subidos.
